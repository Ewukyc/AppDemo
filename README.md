# AppDemo

Android приложение для просмотра IT курсов. Позволяет открывать детальную информацию о каждом курсе, добавлять курсы в избранное и сортировать их по дате публикации.

## Что реализовано

- Тёмная тема с неоновым зелёным дизайном
- Экран входа с валидацией email и пароля
- Главный экран со списком 5 IT курсов
- Карточки курсов с фотографиями и основной информацией
- Детальный экран каждого курса с полным описанием и датой начала
- Поиск (нефункциональный, как по ТЗ)
- Иконка фильтра с сортировкой по дате публикации (убыванию)
- Добавление/удаление курсов в избранное через сердечко
- Раздел "Избранное" с сохранением данных в локальную БД
- Раздел "Профиль" (заглушка)
- BottomNavigation для навигации между разделами

## Технологический стек

- Kotlin
- Android XML layouts (без Compose)
- Room (SQLite) для локального хранилища
- Repository Pattern для работы с данными
- Navigation Component
- Coroutines для асинхронных операций
- Material Design 3 компоненты

## Как запустить

1. Клонируй репозиторий:
```bash
git clone https://github.com/Ewukyc/AppDemo.git
```

2. Открой проект в Android Studio

3. Подключи телефон через USB или запусти эмулятор

4. Нажми Run (Shift+F10)

## Авторизация

На экране входа можешь вводить любой email в формате `text@text.text` и любой пароль. Кнопка "Войти" активируется только когда оба поля заполнены корректно.

Кнопки ВК и OK открывают соответствующие сайты в браузере.

## Особенности реализации

- Фотографии курсов подбираются автоматически по названию
- Избранное сохраняется в локальную Room БД
- Сортировка работает по дате публикации (новые курсы первыми)
- Описание курса на карточке обрезается на 2 строки
- При нажатии на карточку открывается подробная информация о курсе
- На экране деталей видна дата начала курса
